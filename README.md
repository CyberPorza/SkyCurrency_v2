# 🌤️ SkyCurrency v2.0 - Premium Dashboard

SkyCurrency, gerçek zamanlı hava durumu verilerini ve döviz kurlarını tek bir şık panelde toplayan, Python (Flask) tabanlı bir web uygulamasıdır. 

![Status](https://img.shields.io/badge/Status-Live-success) ![Version](https://img.shields.io/badge/Version-2.0-blue)

## ✨ Özellikler
- **Glassmorphism UI:** Modern, karanlık mod ve cam efektiyle tasarlanmış kullanıcı arayüzü.
- **Canlı Döviz:** USD, EUR ve ONS Altın verileri (JSON API entegrasyonu).
- **Hava Durumu:** İstanbul için anlık sıcaklık, nem ve hissedilen sıcaklık bilgileri.
- **Dijital Saat:** JavaScript ile entegre edilmiş, saniye hassasiyetli canlı saat.
- **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu (Bootstrap 5).

## 🛠️ Kullanılan Teknolojiler
- **Backend:** Python 3.x, Flask
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla), Bootstrap 5
- **Veri Kaynakları:** REST APIs (Open Exchange Rates & wttr.in)
- **Deployment:** Render.com
